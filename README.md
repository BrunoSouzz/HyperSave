# HyperSave 🚀

HyperSave é uma aplicação desenvolvida para facilitar o download e gerenciamento de vídeos e áudios a partir de diferentes plataformas, oferecendo uma experiência simples, rápida e intuitiva.

## ✨ Funcionalidades

- Download de vídeos
- Download apenas do áudio
- Conversão automática
- Interface amigável
- Histórico de downloads
- Gerenciamento de arquivos baixados

## 🛠 Tecnologias Utilizadas

- Node.js
- Express
- Vue.js
- Tailwindcss

> Ajuste esta seção conforme seu projeto.

## 📦 Instalação

### Clonando o projeto

```bash
git clone https://github.com/BrunoSouzz/HyperSave.git
cd HyperSave
```

### Instalando dependências

```bash
flutter pub get
```

ou

```bash
npm install
```

## ▶️ Executando o projeto

### Desenvolvimento

```bash
flutter run
```

ou

```bash
npm run dev
```

### Produção

```bash
flutter build apk --release
```
